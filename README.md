# na
Núcleo Académico de la Maestría en Ciencia de Datos

## Sitio Web en GitHub Pages

Este repositorio contiene el sitio web del núcleo académico de la Maestría en Ciencia de Datos de la Universidad de Sonora.

### Activar GitHub Pages

Para activar el sitio:
1. Ve a Settings → Pages
2. En "Source", selecciona "Deploy from a branch"
3. Selecciona la rama principal y guarda
4. El sitio estará disponible en: `https://mcd-unison.github.io/na/`

### Actualizar Información

- **Fotos de profesores**: Reemplaza los archivos en `assets/images/` con fotos reales
- **Información de profesores**: Edita el archivo `index.md`
